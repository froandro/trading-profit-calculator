# Калькулятор сделки

Калькулятор прибыльности сделок с ценными бумагами. Учитывает чистую цену, НКД (накопленный купонный доход), комиссию брокера и налог. Показывает точку безубыточности и цену откупа после снижения.

## Возможности

- Расчёт количества бумаг исходя из суммы инвестиций
- Отдельные поля НКД для покупки, продажи и откупа
- Комиссия брокера на обе стороны сделки
- Налог на прибыль (ставка настраивается, 0–50%)
- Точка безубыточности — при какой цене продажа будет в ноль
- Сценарий откупа после снижения цены
- Две колонки результатов: за 1 шт и всего (× количество)
- Тёмная/светлая тема (кнопка в правом верхнем углу, запоминается)

## Использование

Открыть `index.html` в браузере или перейти на [trading-profit-calculator.vercel.app](https://trading-profit-calculator.vercel.app).

## Разработка

Один файл — `index.html` (HTML + CSS + JS). Никаких сборщиков, зависимостей или сервера.

```bash
# просто открой в браузере
open index.html
```

## Деплой

При пуше в `main` Vercel пересобирает автоматически.
